## 📐 画像の要件

### サイズ

- **推奨**: 1920×1080px（16:9）
- **最小**: 1280×720px
- **最大**: 3840×2160px

### フォーマット

- **推奨**: JPEG (.jpg)
- **代替**: PNG (.png) / WebP (.webp)

### ファイルサイズ

- **推奨**: 各 200KB〜500KB
- **最大**: 1MB 以下（ページ読み込み速度のため）

### 画質

- **解像度**: 高解像度（鮮明な画像）
- **圧縮**: 適度に圧縮（画質とファイルサイズのバランス）

## 🎨 デザインのポイント

各スライドには暗いオーバーレイ（`rgba(0, 0, 0, 0.3)`〜`rgba(0, 0, 0, 0.4)`）が適用されるため：

- ✅ **明るい画像**を使用することを推奨
- ✅ **コントラストが高い**画像が見やすい
- ✅ **中央に主題**がある画像が効果的
- ❌ すでに暗い画像は避ける
- ❌ 過度に複雑な背景は避ける

## 🔧 トラブルシューティング

### 画像が表示されない場合

1. **ファイル名を確認**

   - 正確に `hero_main.jpg` などの名前になっているか
   - 大文字・小文字が正しいか（すべて小文字）

2. **ファイルパスを確認**

   - `images/` ディレクトリ直下に配置されているか
   - パーミッション設定は適切か（644 推奨）

3. **ファイル形式を確認**
   - JPEG 形式（.jpg）になっているか
   - ブラウザキャッシュをクリアしてみる

### 画像の最適化

画像ファイルサイズが大きい場合は、以下のツールで最適化できます：

- [TinyJPG](https://tinyjpg.com/) - オンライン画像圧縮
- [ImageOptim](https://imageoptim.com/) - Mac 用アプリ
- [RIOT](https://riot-optimizer.com/) - Windows 用アプリ

## 📝 現在の設定

各スライドの背景画像は以下の CSS で適用されています：

```css
background-image: linear-gradient(rgba(0, 0, 0, 0.4), rgba(0, 0, 0, 0.4)),
  url("images/hero_main.jpg");
```

- `linear-gradient`: 暗いオーバーレイを追加
- テキストの視認性を向上
- 統一感のあるデザイン
